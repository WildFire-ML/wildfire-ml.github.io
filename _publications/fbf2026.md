The Wildfire Intelligence and Simulation Engine (W.I.S.E.) is a wildfire growth simulation model based on the Canadian Fire Behaviour Prediction (FBP) system, a component of the Canadian Forest Fire Danger Rating System (CFFDRS). This tool is used by wildfire managers in Canada and internationally to predict wildfire growth including intensity, spread rate, and fuel consumption. These characteristics are critical inputs for planning prescribed burns, evacuation routes, and wildfire response activities. 
Satellite remote sensing now enables the creation of fire spread datasets that significantly outnumber traditional ground-based measurements of wildfire spread. The Canadian Fire Spread Dataset (CFSDS) uses satellite-borne infrared sensors to construct daily wildfire spread boundaries for over 3,000 fires across Canada. This substantial increase in data availability enables new models and methods to predict wildfire behaviour.
A set of daily wildfire boundaries from CFSDS are used to train a deep learning model to predict daily wildfire spread. The model extends image segmentation architectures to incorporate both spatial and temporal aspects of wildfire spread using maps of fuels, weather, and topography as inputs. Model accuracy is assessed by comparing fire spread predictions using an independent test set of daily wildfire boundaries from CFSDS; metrics include precision, recall, F1 score, intersection-over-union, and Hausdorff distance. Initial results show that the deep learning model generates wildfire boundaries that spread consistently from the ignition polygon. Daily fire growth predictions from both the deep learning model and W.I.S.E. are assessed against CFSDS daily fire boundaries. W.I.S.E. performance improves with detailed initialization of fire conditions, while the deep learning model operates with minimal user intervention. Results are compared to highlight differences in predictive performance and to discuss the advantages and limitations of machine learning and traditional approaches.
